Social Media API Documentation

Quick reference for the available endpoints.

GET /api/users

Get a paginated list of users.

Query Parameters:

Parameter Type Description
page integer Page number (default: 1)
limit integer Number of items per page (default: 10)
curl -X GET "http://localhost:3000/api/users?page=1&limit=10"

Response

{
  "success": true,
  "count": 2,
  "page": 1,
  "total_pages": 2,
  "pagination": {
    "next": {
      "page": 2,
      "limit": 10
    }
  },
  "data": [
      {
        "id": "1",
        "username": "traveler",
        "full_name": "Karma",
        "profile_picture": "https://example.com/profiles/traveler.jpg",
        "bio": "Travel photographer",
        "created_at": "2023-01-15"
      }
    ]
}

POST /api/users

Create a new user.

Request Body

{
  "username": "new_traveler",
  "email": "new@example.com",
  "full_name": "New Traveler",
  "bio": "Adventure seeker"
}
curl -X POST "http://localhost:3000/api/users" \ -H "Content-Type: application/json" \ -d '{"username":"newuser","email":"newuser@example.com"}'

Response

{
  "success": true,
  "data": {
    "id": "4",
    "username": "new_traveler",
    "email": "new@example.com",
    "full_name": "New Traveler",
    "profile_picture": "default-profile.jpg",
    "bio": "Adventure seeker",
    "created_at": "2023-03-20"
  }
}

PUT /api/users/:id

Update a user by id.

Parameters

Parameter Type Description
id String User ID

Request Body

{
  "bio": "Updated bio",
  "full_name": "Updated Name"
}
curl -X PUT "http://localhost:3000/api/users/1" \ -H "Content-Type: application/json" \ -d '{"bio":"Updated bio"}'

Response

{
  "success": true,
  "data": {
    "id": "1",
    "username": "traveler",
    "full_name": "Updated Name",
    "profile_picture": "https://example.com/profiles/traveler.jpg",
    "bio": "Updated bio",
    "created_at": "2023-01-15"
  }
}

DELETE /api/users/:id

Delete a user by id.

Parameters

Parameter Type Description
id String User ID
curl -X DELETE "http://localhost:3000/api/users/1"

POST /api/posts

Create a new post.

Headers

Header Value Description
X-User-Id 1 ID of the user creating the post

Request Body

{
  "title": "New post",
  "body": "Hello world"
}
curl -X POST "http://localhost:3000/api/posts" \ -H "Content-Type: application/json" \ -H "X-User-Id: 1" \ -d '{"title":"New post","body":"Hello world"}'

Response

{
  "success": true,
  "data": {
    "id": "4",
    "user_id": "1",
    "title": "New post",
    "body": "Hello world",
    "created_at": "2026-03-19"
  }
}

PUT /api/posts/:id

Update a post (requires X-User-Id).

Headers

Header Value Description
X-User-Id 1 User ID (must match post owner)

Request Body

{
  "body": "Updated body"
}
curl -X PUT "http://localhost:3000/api/posts/1" \ -H "Content-Type: application/json" \ -H "X-User-Id: 1" \ -d '{"body":"Updated body"}'

Response

{
  "success": true,
  "data": {
    "id": "1",
    "user_id": "1",
    "title": "Hello World",
    "body": "Updated body",
    "created_at": "2026-03-01"
  }
}

DELETE /api/posts/:id

Delete a post (requires X-User-Id).

Headers

Header Value Description
X-User-Id 1 User ID (must match post owner)
curl -X DELETE "http://localhost:3000/api/posts/1" \ -H "X-User-Id: 1"

Response

{
  "success": true,
  "data": {}
}

GET /api/comments

Get a paginated list of comments.

curl -X GET "http://localhost:3000/api/comments?page=1&limit=10"

POST /api/comments

Create a new comment (requires X-User-Id).

curl -X POST "http://localhost:3000/api/comments" \ -H "Content-Type: application/json" \ -H "X-User-Id: 2" \ -d '{"post_id":"1","body":"Nice post!"}'

DELETE /api/comments/:id

Delete a comment (requires X-User-Id).

curl -X DELETE "http://localhost:3000/api/comments/1" \ -H "X-User-Id: 2"

GET /api/likes

Get a paginated list of likes.

curl -X GET "http://localhost:3000/api/likes?page=1&limit=10"

POST /api/likes

Like a post (requires X-User-Id).

curl -X POST "http://localhost:3000/api/likes" \ -H "Content-Type: application/json" \ -H "X-User-Id: 2" \ -d '{"post_id":"1"}'

DELETE /api/likes/:id

Remove a like (requires X-User-Id).

curl -X DELETE "http://localhost:3000/api/likes/1" \ -H "X-User-Id: 2"

GET /api/followers

Get a paginated list of followers.

curl -X GET "http://localhost:3000/api/followers?page=1&limit=10"

POST /api/followers

Follow a user (requires X-User-Id).

curl -X POST "http://localhost:3000/api/followers" \ -H "Content-Type: application/json" \ -H "X-User-Id: 2" \ -d '{"user_id":"1"}'

DELETE /api/followers/:id

Unfollow a user (requires X-User-Id).

curl -X DELETE "http://localhost:3000/api/followers/1" \ -H "X-User-Id: 2"
